# Aplikasi Klinik Online

## Deskripsi
Aplikasi desktop berbasis C++ yang dikembangkan sebagai implementasi mata kuliah Algoritma Pemrograman Lanjut. Sistem digunakan untuk mengelola data pasien, dokter, jadwal konsultasi, dan pembelian obat.

## Fitur
- Login User
- Login Dokter
- Booking Jadwal
- Data Pasien
- Data Dokter
- Pembelian Obat
- Searching
- Sorting

## Teknologi
- C++
- File Handling
